# Integration Examples

Demonstrates how to integrate codemap into your workflow.

- **claude-code/inject.ts** — Inject the structural map into CLAUDE.md
- **git-hooks/setup.ts** — Install a pre-commit hook to auto-regenerate the map
- **ci-pipeline/github-actions.yml** — GitHub Actions workflow for CI integration

Run: `npx tsx examples/05-integrations/claude-code/inject.ts`

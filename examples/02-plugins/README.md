# Plugin Examples

Demonstrates the codemap plugin system.

- **auto-detect.ts** — How language parsers are loaded automatically by file extension
- **manual-register.ts** — Register plugins manually with `codemap().use()`
- **custom-parser.ts** — Write a custom language parser for any file type
- **custom-formatter.ts** — Write a custom output formatter (YAML example)

Run any example: `npx tsx examples/02-plugins/auto-detect.ts`

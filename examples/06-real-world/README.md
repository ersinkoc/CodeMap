# Real-World Examples

Production-oriented usage patterns.

- **monorepo-scan/** — Scan a monorepo with automatic workspace detection
- **incremental-workflow/** — Use incremental mode for fast re-scans of changed files
- **multi-language/** — Scan a project with TypeScript, Python, Go, Rust, and more

Run: `npx tsx examples/06-real-world/monorepo-scan/scan.ts`
